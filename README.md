# Blockchain Demo

## Project Overview
This project demonstrates a basic implementation of blockchain concepts using Python and Flask. It simulates how data is secured using hashing and how blocks are linked together to form a blockchain.

## Features
- SHA-256 hash generation
- Block creation and mining using Proof of Work
- Visualization of a simple blockchain structure

## Tech Stack
- Python
- Flask
- HTML/CSS

## How to Run
1. Install dependencies:
   pip install flask

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/

## Concepts Covered
- Hashing (SHA-256)
- Proof of Work
- Blockchain structure
- Data integrity

## Purpose
This project is a simplified educational model to understand the fundamental working of blockchain systems.
